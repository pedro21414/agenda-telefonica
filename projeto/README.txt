Este projeto é uma agenda telefônica simples desenvolvida em Python.
O programa permite cadastrar, armazenar e gerenciar nomes e telefones.

O sistema possui um menu onde é possível:

mostrar os contatos cadastrados

cadastrar um novo contato

alterar o telefone de um contato

excluir um contato

buscar um contato pelo nome

favoritar contato

desfavoritar contato




Os contatos são salvos em um arquivo CSV chamado agenda.csv.
Dessa forma, quando o programa é encerrado e aberto novamente,
os dados continuam salvos e não são perdidos.

Quando o programa inicia, ele lê o arquivo CSV e carrega
os dados para listas em memória.
Sempre que algum contato é cadastrado, alterado ou excluído,
o arquivo é atualizado automaticamente.

Arquivos do projeto:
agenda1.py → código principal do programa
agenda.csv → arquivo onde os contatos são armazenados
README.txt → explicação do funcionamento do projeto

No desenvolvimento do código foram utilizados conteúdos estudados em sala,
como listas, manipulação de arquivos, funções, estruturas condicionais
(if/else) e laços de repetição (while).
